# Digimon UP Companion

Aplicación web companion para el juego de cartas **Digimon Card Game**, diseñada para ayudar a los jugadores durante las partidas con herramientas de tracking y utilidades.

## Funcionalidades

### Memory Gauge Helper
- Tablero visual de 24 casillas (4×6) para trackear las posiciones de memoria de los Digimon
- Selección rápida entre 12 Digimon disponibles con iconos visuales
- Historial de acciones con posibilidad de deshacer
- Modo pantalla completa optimizado para jugar en mesa

### Próximas funcionalidades
- Contador de memoria (Memory Gauge) interactivo
- Tracker de seguridad (Security Stack)
- Historial de partida
- Soporte offline (PWA)
- Modo oscuro/claro

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript (vanilla)
- **Hosting**: GitHub Pages
- **Diseño**: Mobile-first, responsive

## Uso

Abre `index.html` en cualquier navegador o visita la versión desplegada en GitHub Pages.

## Desarrollo

```bash
# Clonar el repositorio
git clone https://github.com/branpg/digimon-up-companion.git
cd digimon-up-companion

# Abrir en el navegador
open index.html
```

No requiere instalación de dependencias ni build. Es una aplicación estática pura.
